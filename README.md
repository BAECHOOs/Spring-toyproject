## 📍 사용하는 툴

1. Java : 11 version

2. Spring Boot : 2.4.2

3. Build Tool : Gradle

4. DB : MySQL

5. Deploy : AWS(or GCP)

   











<br><br>

## 📌 역할 분담



### 1) 백엔드 개발에 앞서

1. 순수html, css로 프론트 짜기 (mypage, detail page)

2. Thymeleaf 포맷으로 변환

   (상품 목록은 **th:each** 등)

3. 데이터 베이스 연동 방법

4. 빌드 툴 및 배포 방식 (gradle, maven, jar, war)

5. 로그인 Spring security 사용법 정리
